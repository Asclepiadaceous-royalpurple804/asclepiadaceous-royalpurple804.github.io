📖 Project Overview

The Shop Management System helps store owners organize their stock. Use this desktop application to keep track of items in your retail store or stationery shop. You can store product names, price tags, quantity totals, and categories in one central database. The interface stays simple so you spend less time on paperwork and more time helping customers.

🚀 Getting Started

This software runs directly on your computer. Follow these steps to prepare your machine and start the program for the first time.

1. Check System Requirements

Your computer needs Windows 10 or newer. Ensure you have at least 50MB of free disk space. You do not need expensive hardware for this program to work well.

2. Install Python

This application runs on the Python language. Visit the official Python website to download version 3.10 or newer. Run the installer file after it downloads. Make sure to check the box that says "Add Python to PATH" during the installation process.

3. Downloading the Software

You need the program files to get started. Visit this link to download the software: Shop Management System Download Page. Choose the latest release and save the file to your computer. Create a new folder on your computer named "ShopSystem" and extract all the downloaded files into that folder.

4. Running the Application

Open the folder where you placed the files. Hold the Shift key and right-click on an empty space inside that folder. Choose the option to "Open PowerShell window here" or "Open in Terminal". A blue or black window will appear. Type the following command and press Enter:

python ShopManagmentSystem_DataFlair.py

The main window will appear on your screen after a few seconds.

✨ Key Features

  • Inventory Tracking: Keep a clear record of every item currently in your store.
  • Add New Stock: Input product names and descriptions whenever you receive new shipments.
  • Price Management: Update price tags instantly across your database.
  • Stock Level Monitoring: Watch your quantity counts to know when you need to order more units.
  • Search Function: Locate specific items within your list instantly without scrolling through pages.
  • Easy Removal: Delete records for items you no longer carry in your store.

💡 Frequently Asked Questions

Do I need an internet connection? No. This program stores all data on your local hard drive. You can run it offline at any time.

Is my data safe? Yes. Because the database file stays on your computer, you have total control over your information. We recommend that you copy the database file to a USB drive or cloud storage once a week to prevent data loss.

How many items can I add? The system handles thousands of records. Performance remains fast regardless of the number of products in your list.
